# 01 Requirements

## 目标用户

- 量化研究实习申请者（尤其深度学习策略方向）
- 希望复现“防数据泄露”研究流程的研究员

## 核心需求

1. 金融专用 CV：Purged + Embargo + Walk-forward
2. 统一数据集构建：特征/标签/对齐/mask/企业行动接口
3. 可复现实验：seed + YAML 配置 + 结果落盘
4. 轻量回测：收益、回撤、换手、Sharpe
5. PyQt6 GUI：重型工作台（数据、实验、监控、回测、对比）
6. 聚宽/米筐真实适配：认证、限频、重试、错误处理
7. 高频微观结构特征：盘口不平衡、成交强度、短窗波动、OFI

## 非目标

- 通用 AutoML
- 全市场交易执行系统
- 云原生分布式训练平台
