# 06 GUI PRD

## 目标

提供专业、简洁、美观的研究工作台，支持从配置到结果对比的一站式流程。

## 页面

1. Data：数据源和凭证策略提示
2. Dataset：特征/标签/mask 配置摘要
3. Experiment：配置路径、运行按钮、任务触发
4. Monitor：日志流
   - 展示 `status/failure_reason`（success/blocked/failed）
5. Backtest：指标表
6. Explainability：因子 IC/RankIC/分位收益贡献面板
7. Compare：历史实验排序对比

## 体验原则

- 暗色专业风格
- 高信息密度但不拥挤
- 一键启动 + 可追踪 + 可回看
